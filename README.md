# first-frontend

Frontend de **First**, una plataforma multipropósito (CRM + CMS). SPA en
React + TypeScript + Vite, con arquitectura domain-sliced. Consume la API de
`first-backend`.

## Empezar

```bash
pnpm install
pnpm dev
```

Otros comandos: `pnpm typecheck`, `pnpm lint`, `pnpm lint:sonar`,
`pnpm vitest run`, `pnpm build`.

## Convenciones

Ver [`CLAUDE.md`](./CLAUDE.md) para arquitectura, comandos y convenciones de
componentes (qué va en `shared/ui`/`shared/lib`, campos numéricos, etc.).
